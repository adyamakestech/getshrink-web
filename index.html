Bagaimana Cara Kerjanya?

Kebanyakan situs kompresi PDF mengharuskan Anda mengunggah file ke server mereka. Hal ini memakan waktu (terutama jika file besar) dan menimbulkan risiko keamanan data.

GetShrink menggunakan teknologi JavaScript modern. Saat Anda memilih file, browser Anda sendirilah yang bekerja "membongkar" PDF tersebut, mengecilkan ukuran gambarnya, dan merakitnya kembali. Semuanya terjadi secara lokal.

Privasi 100% Terjamin

  • Tanpa Upload: File Anda tidak pernah dikirim ke internet.
  • Tanpa Batas Ukuran: Karena menggunakan perangkat Anda, tidak ada batasan file maksimal.
  • Aman & Cepat: Proses lebih cepat karena tidak menunggu proses upload & download dari server.

Pertanyaan yang Sering Diajukan (FAQ)

Kenapa ukuran PDF saya malah menjadi lebih besar?

Metode yang kami gunakan (Rasterisasi) sangat optimal untuk PDF hasil jepretan kamera (scan) atau yang penuh gambar. Jika PDF Anda murni berisi teks digital, metode ini bisa membuat ukuran file sedikit membengkak.

Apakah aplikasi ini gratis?

Ya, 100% gratis tanpa batasan penggunaan. Kami mendanai operasional website ini melalui iklan yang ditampilkan.
